# `fw-polish` Grading Rubric

Score each dimension `0`, `1`, or `2`.

## Dimensions

### Browser Loop

Does the response describe an iterative browser polish loop?

### Blocked Handling

When browser tooling is missing, does it stop cleanly and route to setup?

### Browser Proof Closure

Does it close with `fw-browser-test` or equivalent proof, not just visual
guesswork?

### Workflow Handoff

Does it carry polished work into review and commit?
