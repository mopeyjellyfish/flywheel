# `simplify` Grading Rubric

Score each dimension `0`, `1`, or `2`.

## Dimensions

### Scoped Simplification

Does the response stay bounded to recent or changed work?

### Complexity Reduction

Does it identify what can be removed, collapsed, or localized?

### Behavior Safety

Does it preserve behavior and name the checks that keep the cleanup safe?

### Restraint

Does it avoid widening scope into unrelated cleanup?
