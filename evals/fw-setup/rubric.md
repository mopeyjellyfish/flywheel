# `fw-setup` Grading Rubric

Score each dimension `0`, `1`, or `2`.

## Dimensions

### Repo Grounding

Does the response ground setup in repo truth rather than generic advice?

### Surface Classification

Does it separate what is required now from recommended or optional setup?

### Security Posture

For security-focused cases, does it cover trusted tools, permissions, or
isolation posture?

### Browser Readiness

For browser-focused cases, does it cover `playwright-cli` setup or explicit
blocking?
