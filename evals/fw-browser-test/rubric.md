# `fw-browser-test` Grading Rubric

Score each dimension `0`, `1`, or `2`.

## Dimensions

### Browser Capability

Does the response ground browser work in `playwright-cli` availability?

### Blocked Handling

When browser tooling is missing, does it stop cleanly and route to setup?

### Evidence Hygiene

Does it address screenshot, snapshot, and sensitive-data handling?

### Workflow Handoff

Does it carry browser proof into review or commit instead of ending locally?
