# `logging` Grading Rubric

Score each dimension `0`, `1`, or `2`.

## Dimensions

### Current Truth

Does the response begin from the existing logging setup?

### Event Model

Does it define useful event names, fields, identifiers, or outcomes?

### Sensitive Data Discipline

Does it protect secrets, tokens, passwords, and other sensitive data?

### Operability

Does it connect logging to real debugging and operator needs?
