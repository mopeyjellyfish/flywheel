# `fw-worktree` Grading Rubric

Score each dimension `0`, `1`, or `2`.

## Dimensions

### Script Usage

Does the response route through the bundled worktree manager or supported
worktree surface?

### Env Safety

Does it preserve the env-file safety posture?

### Action Routing

Does it select a supported action such as create, list, path, copy-env, or
cleanup?

### Next Step

Does it give the user a concrete next step?
