# `document-review` Grading Rubric

Score each dimension `0`, `1`, or `2`.

## Dimensions

### Document Targeting

Does the response clearly review a document path or document type?

### Reviewer Selection

Does it apply concrete reviewer lenses such as scope, feasibility, security,
observability, or simplicity?

### Findings Structure

Does it present findings, verdicts, severity, or another review-shaped output?

### Mode Handling

When `mode:headless` appears, does it respect headless constraints?

### Workflow Fit

Does it route the reviewed document into the correct downstream workflow?
