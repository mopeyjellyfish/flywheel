# `flywheel-runtime-change` Grading Rubric

Score each dimension `0`, `1`, or `2`.

## Dimensions

### Earliest Stage

Does the response start at the correct point in the runtime-change journey?

### Journey Coherence

Does it preserve the expected multi-stage path rather than collapsing it into a
single stage?

### Risk Gating

Does it keep runtime risk, rollout, validation, or rollback posture visible?

### Artifact Carry

Does it carry forward plan, review, evidence, or rollout artifacts across the
journey?

### Closure

Does it keep the journey pointed toward commit and final lesson capture?
