# `flywheel-incident-response` Grading Rubric

Score each dimension `0`, `1`, or `2`.

## Dimensions

### Earliest Stage

Does the response start at incident handling rather than skipping straight into
debug or commit?

### Journey Coherence

Does it preserve a coherent multi-stage incident path?

### Evidence Preservation

Does it keep runtime evidence and incident artifacts alive across the journey?

### Risk Gating

Does it keep mitigation, rollback, patch, or rollout decisions explicit?

### Closure

Does it still point the journey toward safe commit and final lesson capture?
