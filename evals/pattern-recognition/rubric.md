# `pattern-recognition` Grading Rubric

Score each dimension `0`, `1`, or `2`.

## Dimensions

### Repo Pattern Grounding

Does the response start from repo patterns or clearly state when the repo has
no strong pattern yet?

### Pattern Fit

Does it map the actual problem signal to the right pattern family or to no
pattern at all?

### Anti-Pattern Discipline

Does it explain what should not be added and why?

### Restraint

Does it avoid cargo-cult pattern adoption and prefer the simpler local design
when appropriate?
