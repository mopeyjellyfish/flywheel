# `observability` Grading Rubric

Score each dimension `0`, `1`, or `2`.

## Dimensions

### Platform Grounding

Does the response ground observability in repo truth or the actual platform in
use?

### Runtime Surface

Does it identify the runtime surface being instrumented?

### Signal Plan

Does it define concrete logs, traces, metrics, dashboards, or validation?

### Choice Surface

When there are real options, does it present a recommendation plus sensible
alternatives?
