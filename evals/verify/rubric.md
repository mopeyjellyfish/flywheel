# `verification-before-completion` Grading Rubric

Score each dimension `0`, `1`, or `2`.

## Dimensions

### Claim Definition

Does the response define the claim being verified?

### Proof Identification

Does it identify the exact proof path, command, query, or artifact?

### Fresh Verification

Does it require a fresh result from the current tree?

### Honest Status

Does it distinguish real evidence from wishful thinking and return a truthful
status?
